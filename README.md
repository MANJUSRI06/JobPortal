# Jobify - Job Finder Application

Jobify is a modern, responsive web application built with React and Redux that simplifies the job search process. It allows users to browse job listings, view detailed job descriptions, and explore company profiles in a seamless and intuitive interface.

- **Interactive UI**: Smooth navigation and transitions powered by React Router.
- **State Management**: Efficient data handling using Redux Toolkit.
